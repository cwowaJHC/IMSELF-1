Briefly — how it will work (scenario) The user logs in → sees the search bar “Describe your problem” (natural language) or selects a category/subcategory. The search recognizes the problem → creates a “PROBLEM” card (tab). Tabs are like in a browser: you can switch, drag, and pin them. The main tab shows: recommended practices (at the beginning), books, videos, applications, and a plan (AI planner).

Коротко — как это будет работать (сценарий) Пользователь заходит → видит строку поиска «Опиши свою проблему» (натуральный язык) или выбирает категорию/подкатегории. Поиск распознаёт проблему → создаёт карточку «PROBLEM» (вкладка). Вкладки — как в браузере: можно переключать, перетащить, закрепить. Главная вкладка показывает: рекомендуемые практики (в начале), книги, видео, приложения и план (AI-планировщик). Пользователь добавляет практику/задачу в своё расписание → система отслеживает выполнение → обновляет прогресс и даёт награды. UI/UX — идеи и элементы (чётко по твоему макету) Вкладки PROBLEM (вверху) Внешний вид: пилы/пилюли (скруглённые табы) как в новых браузерах. Активная — самая большая по центру. Функции таба: название проблемы, иконка категории, краткий прогресс (маленький круглый индикатор). Управление: drag&drop для порядка, двойной клик — переименовать, клик правой кнопкой/3-точки — контекстное меню (см. ниже). Поисковая строка NLP-поле: пользователь пишет проблему (напр. «не могу вставать по утрам», «не хватает мотивации учиться») → система парсит теги, предлагает быстрые фильтры («здоровье», «привычки», «мотивация»). Подсказки: автодополнение, похожие готовые проблемы (шаблоны). Левая кнопка (слева от логотипа) Открывает боковую панель (sidebar) с категориями, избранным, быстрыми практиками, AI-помощником. Три точки (справа) на верхней панели Меню: Настройки профиля / Уведомления / Экспорт (PDF) / Поделиться проблемой / Скопировать вкладку / Закрепить / Закрыть все / Фидбек. Нижний колонтитул Авторские права, контакты, политики (конфиденциальность), ссылки на Telegram/YouTube/FAQ, кнопка «Сообщить о проблеме/ошибке». Система прогресса — дизайн + формула Предлагаю комбинированную систему: привычки + время практик + выполнение задач + субъективная оценка. Метрика «Progress Score (0–100%)»: Habits completion (выполненные привычки / запланированные) — вес 40% Practice time (минуты занятий / цель минут) — вес 30% Challenges & tasks (выполнение заданий) — вес 20% Self-rating (10-сбалльный опрос «как ты себя ощущаешь») — вес 10% Формула: Progress = 0.4*Habits% + 0.3*PracticeTime% + 0.2*Tasks% + 0.1*SelfRating% Пример расчёта (цифры шаг за шагом): Habits = 80% → 0.4 * 80 = 32 PracticeTime = 60% → 0.3 * 60 = 18 Tasks = 50% → 0.2 * 50 = 10 SelfRating = 70% → 0.1 * 70 = 7 Итого Progress = 32 + 18 + 10 + 7 = 67% Визуализация прогресса Большой прогресс-бар на карточке проблемы + «Skill tree» (дерево навыков) с уровнями mastery (Beginner → Intermediate → Advanced). История прогресса: график по дням + недельные и месячные достижения. «Confidence meter» — шкала настроения (связана со Self-rating). AI-помощник — возможности Разбор запроса: парсит проблему, предлагает 3 уровня решения: быстрые практики (5–10 минут), курс (2–4 недели) и глубокая методика. Генерация расписания: один клик — вставить практику сразу в день/неделю; экспорт в Google Calendar / PDF. Подбор материалов: книги, видео, приложения, чек-листы. Мини-коучинг: диалог-бот, который задаёт уточняющие вопросы и предлагает шаги. Шаблоны вопросов: «Дай 3 простые практики на утро», «Составь план на 21 день для привычки X». Ограничения: в бесплатной версии — 3 запроса в день; в премиум — неограниченно. Геймификация & Социальное Очки / энергия / XP — за выполнение практик, прохождение челленджей, ежедневные streak’и. Badges (значки) — за streak 7/30/100 дней, за 5 пройденных курсов и т.д. Ранги — Новичок → Исследователь → Создатель → Мастер. Ранги открывают фичи (например, доступ к закрытым челленджам). Leaderboards — в сообществе, по друзьям, по категории. Social sharing — пост о достижении в соцсети (с шаблоном картинки). Командные челленджи — можно объединяться и соревноваться. Монетизация — планы (включая $4.99) Рекомендую три уровня: Free (бесплатно) Ограниченный каталог (часть книг/аудио заблокированы) 3 AI-запроса/день Ограничение на количество активных вкладок/проблем (например, 3) Базовая статистика и магазин с платными товарами Premium — $4.99 / месяц (твоя основная платная версия) Неограниченные AI-запросы Неограниченные вкладки и шаблоны проблем Полный доступ к библиотеке практик и большинству аудиокниг Экспорт PDF/CSV, кастомные темы интерфейса +10% скидка в магазине, премиум-бейджи Creator / Pro — $7–10 / месяц (или по достижении очков) Всё из Premium + персональный AI-планировщик (детализированный план на 3 месяца) Доступ к «Проектам» (инструменты для запуска собственных проектов) Приоритетная поддержка и вебинары Дополнительно: внутренняя валюта (очки) для покупки материалов/скидок; одноразовые покупки. Adaptive difficulty: если пользователь не выполняет — AI упрощает задачи; если успешно — увеличивает сложности. Offline mode: сохранять практики, трекер и читать/слушать офлайн.
